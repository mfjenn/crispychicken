= Project Crispy Chicken Sandwich

A website that allows users to create a customizable event map and allows its users to share a kickass front end UI for them share with their friends/family/colleagues. 

We live in an age where technology drives us. We will use what the internet already provides to create a tool for people to create events, display locations and customize it to their hearts content.

=== Alpha Release:
* Create/Login User Account 
* Create/Delete Events 
* View your own event with Google Maps Generated
